# 高仿开眼
  ---
- 用到的第三方库
  - GuillotineMenu   : 刀砍式转场动画
  - Alamofire        : 网络请求库
  - APESuperHUD      : Swift的一个HUD
  - SnapKit          : Swift版的Masonry,屏幕适配库
  - SwiftyJSON       : Swift的JSon解析库
  - YYWebImage       : 图片加载库
